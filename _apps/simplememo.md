📝 シンプルで使いやすいWeb上で動作する[メモ帳アプリ](https://takiya21.github.io/WebAppTest/)です。ブラウザで快適にメモを作成・編集・削除できます。

## 🚀 アクセス方法

以下のリンクをクリックしてアクセスしてください：

[**シンプルメモ帳アプリを開く** →](https://takiya21.github.io/WebAppTest/)


## ✨ 機能

- **メモ作成**: 「+ 新規メモ」ボタンでいつでも新しいメモを作成
- **メモ編集**: 「編集」ボタンでメモを編集可能
- **メモ削除**: 「削除」ボタンで不要なメモを削除
- **自動保存**: メモはブラウザのローカルストレージに自動保存（ブラウザを閉じてもデータが残ります）
- **レスポンシブデザイン**: パソコンとスマートフォンの両方で快適に使用可能

## 📖 使い方

1. `index.html` をブラウザで開く
2. 画面上部の「+ 新規メモ」ボタンをクリック
3. テキストエリアにメモを入力
4. 「保存」ボタンをクリック
5. メモ一覧に保存されたメモが表示されます

### その他の操作

- **再度編集する**: メモの「編集」ボタンをクリック
- **メモを削除する**: メモの「削除」ボタンをクリック
- **キーボードショートカット**: 編集中に `Ctrl + Enter`（またはMacでは `Command + Enter`）で保存

## 📁 ファイル構成

```
WebAppTest/
├── index.html   - HTML構造
├── style.css    - スタイル（見た目）
├── script.js    - 機能（メモの作成・編集・削除など）
└── README.md    - このファイル
```

## 💾 データ保存について

メモはブラウザのローカルストレージに保存されます。
- データはあなたのコンピュータに保存されます
- ブラウザのキャッシュを削除するとデータが失われる可能性があります
- クラウドバックアップには対応していません
