レイク浜松カントリークラブによく行くので、レイク浜松カントリークラブ専用のスコア管理アプリ「**Golfys**」を作成しました。

## アプリの特徴

ブラウザで動作するゴルフスコア管理ツール。スコアをデバイスに保存し、ホールごとの分析でラウンドを振り返ることができます。

## 主な機能

### 📊 メインページ（ダッシュボード）

- **スコア推移グラフ**: 全ラウンドのスコアとパット数を時系列で表示
  - 棒グラフで視覚的に推移を確認
  - 合計スコア（棒の高さ）= パット数 + スコア（パット除外）
  
- **基本統計サマリー**: 6つの主要指標を一覧表示
  - 総ラウンド数、平均スコア、平均パット数
  - FWキープ率、ベストスコア、プレーコース数

- **クイックアクション**: スコア登録と分析機能への素早いアクセス

### ✍️ スコア手動入力ページ

- コース選択（レイク浜松カントリークラブ）
- プレー日の指定
- 18ホール分の詳細情報入力
  - スコア・パット数
  - 第1打クラブ
  - フェアウェイキープ状況
  - OB数・バンカー数・ペナルティ数
- デバイスに自動保存

### 🏌️ コース選択ページ

- 18ホールをグリッド表示
- 各ホールの画像（webp形式）を確認
- ホール別の平均スコア・パット数を表示
- ホール詳細ページへの遷移

### 🔍 スコア分析ページ

各ホールの詳細分析が可能：

**スコア別成績表**
- バーディ以下、パー、ボギー、ダブル、トリプル、＋４以上の成績分布

**成績統計**
- パーオン率、ボギーオン率
- OB、バンカー、ペナルティの発生率

**FW統計**
- フェアウェイキープ率
- 左右へのズレ、ショートの割合

**ホール詳細画像**
- 大きく表示されたホール画像で目視確認可能

**プレー履歴**
- 日付、スコア、パット数などの過去データを一覧表示

**スコア分布グラフ**
- パイチャートでスコア別の構成比を可視化

## 📋 記録できる情報

各ラウンドで記録される詳細データ：
- コース名
- プレー日時
- ホール番号
- スコア・パット数
- 第1打クラブ
- フェアウェイキープ（〇/左/右/ショート）
- トラブル統計（OB数、バンカー数、ペナルティ数）

## 💾 データ保存について

- **保存方式**: ブラウザのIndexedDB（ローカルストレージ）
- **インターネット不要**: オフライン環境で完全動作
- **データ容量**: ブラウザ容量内で数GB まで保存可能
- **プライバシー**: すべてのデータはデバイスに保存、外部送信なし

## 🎨 ユーザーインターフェース

- **レスポンシブデザイン**: スマートフォン・タブレット・PCで最適化
- **カラースキーム**: グラデーション（紫・ピンク系）で視覚的に統一
- **直感的な操作**: ゴルフプレイヤー向けに最適化されたナビゲーション

## 🔧 技術スタック

- **フロントエンド**: React 18 + Vite
- **ルーティング**: React Router
- **グラフ表示**: Recharts
- **ビルド・デプロイ**: GitHub Pages + npm
- **データベース**: IndexedDB（ブラウザ内）

## 📝 使用例

1. メインページから「スコアを手動入力」をクリック
2. レイク浜松を選択し、プレー日を指定
3. 18ホール分のスコアを入力（パット数やFW状況も記録）
4. 登録完了
5. メインページでスコア推移をグラフで確認
6. コース分析で各ホールの詳細データを閲覧
